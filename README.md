# Personal Website

This is my personal website showcasing my skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy Navigation**: Quick access to all sections with smooth scrolling
- **Sections Include**:
  - Home/Hero section with introduction
  - About section with personal information
  - Skills section highlighting technical abilities
  - Projects section showcasing work
  - Contact section with social links

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

Simply open `index.html` in your web browser to view the website locally.

## Customization

Feel free to customize the content by editing:
- `index.html` - Update text content and structure
- `styles.css` - Modify colors, fonts, and layout
- `script.js` - Add or modify interactive features
